🔥 FIREBASE SETUP REQUIRED 🔥

Your app has been migrated to Firebase Realtime Database!

This means players can now register from different devices and locations.
All data is shared in real-time across all users.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  ACTION REQUIRED BEFORE RUNNING THE APP:

1. Create a Firebase project (5 minutes)
   → Go to: https://console.firebase.google.com/
   → Click "Add project"
   → Name it "cricket-player-app"
   → Create a Realtime Database in Test Mode

2. Get your Firebase credentials
   → Project Settings → Your apps → Web app (</> icon)
   → Copy the firebaseConfig object

3. Create a .env file in the project root
   → Copy .env.example to .env
   → Paste your Firebase credentials

4. Run the app
   → npm run dev
