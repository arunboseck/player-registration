# Public Player Registration

## 🔗 Public Registration URL

Players can register themselves without needing to login to the system using this URL:

### Local Development
http://localhost:5173/register

### After Deploying to Vercel
https://your-app-name.vercel.app/register

## 📋 How It Works

1. Share the Link: Send the /register URL to players who need to register
2. No Login Required: Players can access this page directly without any authentication
3. Self Registration: Players fill in their own details
4. Instant Submission: Data is saved immediately to the system
5. Success Confirmation: Players see a success message after registration

## Key Features

✅ No Login Required - Public access for players
✅ Mobile Friendly - Works perfectly on phones
✅ Form Validation - All fields are validated
✅ Photo Upload - Players can upload their photo
✅ Success Screen - Clear confirmation after registration
✅ Multiple Registrations - Can register multiple players
